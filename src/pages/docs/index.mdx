# Scrap Mechanic Modding Documentation

Welcome to the official ReDoIng Mods documentation. This guide will help you get started with modding Scrap Mechanic and creating amazing content for the game.

## What is Scrap Mechanic?

Scrap Mechanic is a creative multiplayer sandbox game that allows players to build vehicles, structures, and contraptions using a variety of parts and blocks. The game features a robust physics system and offers plenty of opportunities for creative expression through modding.

## Why Mod Scrap Mechanic?

Modding Scrap Mechanic allows you to:

- Add new items, blocks, and parts to the game
- Create custom gameplay mechanics
- Enhance the existing game features
- Share your creations with the community
- Collaborate with other modders

## Getting Started

If you're new to modding Scrap Mechanic, we recommend starting with our [Getting Started](/docs/getting-started) guide, which covers the basic tools and concepts you'll need to know.
